# 🎮 포로봇(PoroBot) 사용 설명서

이 문서는 개발자가 아닌 **관리자(내전 운영진)**와 **플레이어**를 위한 사용 설명서입니다.
복잡한 설치 과정 없이, 봇을 어떻게 사용하는지에 대해서만 다룹니다.

---

## 🏆 플레이어용 명령어
누구나 사용할 수 있는 명령어입니다.

### 1. 랭킹 확인하기
현재 서버의 내전 MMR 순위(Top 10)를 확인합니다.
- **명령어**: `/랭킹` (또는 `!랭킹`)
- **설명**: 1위부터 10위까지의 플레이어 이름, 티어, 현재 MMR, 승률을 보여줍니다.
- **티어 시스템**:
  - **SS**: 1600+
  - **S**: 1400+
  - **A**: 1200+
  - **B**: 1000+
  - **C**: 1000 미만

---

## 👑 관리자용 명령어
내전을 진행하고 결과를 기록하는 운영진 전용 명령어입니다.
*(서버 관리 권한이 있는 계정만 사용할 수 있습니다)*

### 1. 팀 등록하기 (경기 준비)
내전을 시작하기 전, 두 팀을 등록하는 단계입니다.

- **명령어**: `/팀등록`
- **입력 항목**:
  - `team_name`: 팀 이름 (예: `1팀`, `블루팀`)
  - `member1` ~ `member6`: 팀에 속한 플레이어를 멘션(@아이디)으로 선택
  - `seed` (선택): 초기 MMR (기본값 0, 보통 비워두면 됩니다)

> **💡 팁**: 
> - 한 팀에 최대 6명까지 등록 가능합니다.
> - 5:5 내전이라면 `member1`부터 `member5`까지만 채우면 됩니다.

### 2. 경기 결과 기록하기
경기가 끝나면 승패를 기록하여 MMR을 변동시킵니다.

- **명령어**: `/기록`
- **입력 항목**:
  - `team_a`: A팀 멤버들의 ID (띄어쓰기로 구분)
  - `team_b`: B팀 멤버들의 ID (띄어쓰기로 구분)
  - `winner`: 승리한 팀 (`A` 또는 `B` 입력)

> **⚠️ 주의**: 
> - `/팀등록`은 편의 기능이며, 실제 기록은 `/기록` 명령어로 한 번에 처리합니다.
> - **ID 입력 방법**: 채팅창에 멘션(@)을 하면 숫자로 된 ID가 아닌 이름이 보일 수 있습니다. 
>   - 가장 쉬운 방법은 **PC 버전**에서 사용자 우클릭 -> **ID 복사**를 사용하는 것입니다.
>   - 또는 `/팀등록`을 사용했을 때 봇이 보여주는 멤버 목록을 참고하세요.

### 3. 봇 상태 확인
- **명령어**: `/디버그`
- **설명**: 봇이 정상적으로 작동 중인지, 데이터베이스에 몇 명의 플레이어가 저장되어 있는지 확인합니다.

## ❓ 자주 묻는 질문 (FAQ)

**Q. MMR은 어떻게 오르나요?**
A. 이기면 +25점, 지면 -25점이 됩니다. (현재 버전 기준)

**Q. 팀 밸런스를 맞춰주는 기능은 없나요?**
A. 현재 버전에는 자동 밸런싱(`/balance`) 기능이 포함되어 있지 않습니다. 수동으로 팀을 짜주셔야 합니다.

**Q. 전적 검색(`/history`)은 안 되나요?**
A. 현재 버전에서는 개인 전적 조회 기능이 활성화되어 있지 않습니다. `/랭킹`을 통해 요약된 정보만 볼 수 있습니다.
